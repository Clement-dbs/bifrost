// Features

Everything you need.
Nothing you don't.

Built for developers and small teams who just want their data flowing — without the DevOps overhead.

⚡
Any HTTP Source

If an API returns JSON, it's a source. No connectors to write, no schemas to define. Paste a URL, configure auth, done.

⏱
Smart Scheduling

From real-time to daily. Set thresholds, get alerts when collections fail. APScheduler under the hood — lightweight and reliable.

📦
Multi-destination Export

CSV, JSON, Parquet locally. S3, PostgreSQL, BigQuery for production. One toggle in the UI.

🔐
Auth Support

API keys, Bearer tokens, Basic auth, OAuth 2.0. Credentials stored securely per user, never shared.

🖥
Web UI + CLI

A clean no-code interface for the browser. A full CLI for power users — bifrost start, bifrost sources list.

// How it works

From zero to collecting
in four steps.

No infra, no YAML pipelines, no DAGs. Just a Python package.

01
Install

One command, no dependencies to manage manually.

02
Init

Creates your local SQLite database. Ready in seconds.

03
Configure

Add your API sources in the UI. Set auth, format, schedule, and export destination.

04
Collect

Bifrost handles the rest. Data flows automatically to your chosen destination.
